# Model3UI-Snaps-Project
A collection of remastered snapshots for Sega Model 3 UI for 20240128+ by @BackPonBeauty

## About:
Model3UI-Snaps-Project is a project aimed for remaster the snapshots for the UI for Supermodel 20240128+ by @BlackPonBeauty. It has more than 40 pictures in each folder in 2 formats, plus an extra folder that works as a backup of the original pictures from the UI.

## How to use:
Just select a folder (clones, titles + title_screens), copy and paste the images onto the "snaps" folder from the UI program.

### That's it
This repository will rarely update if there are more games that are executable in the emulator, or even listed in the UI. So bye for the moment.
